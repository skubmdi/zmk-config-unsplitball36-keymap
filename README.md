## ball scroll
- Vertical scrolling when there is only function layer
- Horizontal scrolling when there is only number layer

## base layer
Tapping the base key switches the base layer. The valid tapping term is 1000ms.
- One tap : qwerty
- Two tap : dvorak
- Three tap : colemak

## hold-tap Interrupt Flavor
- lt : hold-preferred
- mt : hold-preferred, quick-tap-ms is 300 (You can long-press a key by tapping twice within this number of seconds)

## keymaps
The actions displayed on multiple lines in the standard keymap are tap dances.

Pressing the key multiple times within a set time will change its behavior.
